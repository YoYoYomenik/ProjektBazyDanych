# ZalBD
Projekt GUI wraz z bazą danych na studia.
Tabele i rekordy bazy danych pochodzą z książki autorstwa Bena Forta "SQL w mgnieniu oka".
Zdaję sobię sprawę, że GUI mogłoby być bardziej "ogólne" i obsługiwać różne bazy danych zamiast tej jednej konkretnej, ale nie widzę powodu dla którego miałbym poświęcać czas na robienie drugiego Accessa.
